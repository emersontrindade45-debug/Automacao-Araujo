# N8n Flows — Araujo Hub Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Criar os 5 workflows N8n com IA (GPT-4o) para automação completa de atendimento WhatsApp via Evolution API, incluindo classificação de intenção, fechamento de pedido multi-turno, handoff para humano, follow-up cron e notificação de preço.

**Architecture:** A Evolution API (`evo.evoapi.shop`) recebe mensagens WhatsApp e envia webhooks diretamente ao N8n. O N8n processa com GPT-4o, responde ao cliente via Evolution API, e devolve handoffs estruturados ao Hub Next.js via `POST /api/webhooks/n8n`. O estado multi-turno do Fluxo 2 é mantido em um Data Table N8n com chave por telefone.

**Tech Stack:** N8n 2.51.1 em `https://n8n.evoapi.shop`, Evolution API v2 em `https://evo.evoapi.shop`, OpenAI GPT-4o, Supabase REST API, n8n-mcp MCP tools para criação via API.

## Task 4: Fluxo 2 — Fechamento de Pedido
